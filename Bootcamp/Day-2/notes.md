# Class - 02

## C Tocken 

- C প্রোগ্রামে ব্যবহৃত মৌলিক বিষয়
- C প্রোগ্রামিং এ মোট ৬ টি token রয়েছে।
    1. Keywords (int, float, if, while)
    2. Identifiers (name of  variables and functions)
    3. Constant (10, 3.14, A)
    4. Strings ("Hello")
    5. Operators (+, -, *, /)
    6. Special Symbols / Puncuators( (), <>, {}, #, ., ; ) 

## Data Type

- প্রোগ্রাম এ কোন ধরনের মান store করা হবে, তা নির্ধারন করে।
- Primary Data Types: int, float, char, double
- Derived Data Types: array, pointer, structure

## Format Specifier
    1. int    -   %d
    2. float  -   %f
    3. char   -   %c
    4. double -   %lf
    5. string -   %s

## Variable
- এমন নাম যার মাধ্যমে আমরা ডেটা সংরক্ষণ করি ও ব্যাবহার করি।
- নিয়মঃ নাম অক্ষর /আন্ডারস্কোর দিয়ে শুরু, স্পেস/বিশেষ চিহ্ন না থাকা।
- Example: age, _age, is_registered

## Operators
- গাণিতিক বা যৌক্তিক কাজ করার জন্য ব্যবহৃত চিহ্ন।
    1. Arithmatic Operators: +, -, *, /, %
    2. Realtional Operators: ==, !+, >, <, >=, <=
    3. Logical Operators: &&, ||, !
    4. Assignment Operators: =, +=, -=, *=, /=
